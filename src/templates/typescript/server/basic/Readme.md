# GenTool: Basic Typescript Server

## Develop

Been in root project directory

```bash
# Go to src (if there is 'src' dir)
cd src

# Install dependencies (if not installed yet)
npm i

# To run
npm run dev
```

## Build

Been in root project folder

```bash
# Go to src (if there is 'src' dir)
cd src

# Install dependencies (if not installed yet)
npm i

# To build
npm run build
```

## Production

Been in root project folder

```bash
# Go to src (if there is 'src' dir)
cd src

# Install dependencies (if not installed yet)
npm i

# To run both server and UI
npm run prod
```
